1 Задание №1: Графы и кратчайшие пути

Основные понятия:

  • Граф - конечное множество точек (вершин), соединенных линиями (ребрами)
  • Вершина - точка графа
  • Ребро - линия, соединяющая вершины
  • Степень вершины - количество ребер, исходящих из вершины
  • Матрица смежности - способ представления графа в виде квадратной матрицы

Типы графов:

  • Неориентированный - ребра не имеют направления
  • Ориентированный - каждому ребру присвоено направление
  • Взвешенный - каждому ребру соответствует значение (вес)

Пример из учебника: Между населенными пунктами А, В, С, D, E, F построены дороги, протяженность которых приведена в таблице. Определите длину кратчайшего пути между пунктами А и В.

Решение:

  1. Находим все возможные пути из A в B
  2. Вычисляем длину каждого пути
  3. Выбираем путь с минимальной длиной

Ответ: 2

4 Задание №4: Кодирование информации

Основные понятия:

  • Условие Фано - ни одно кодовое слово не является началом другого
  • Обратное условие Фано - ни одно кодовое слово не является концом другого
  • Алгоритм Хаффмана - метод построения префиксного двоичного кода минимальной длины

Алгоритм Хаффмана:

  1. Создать для каждого символа узел с его частотой появления
  2. Поместить узлы в приоритетную очередь, отсортированную по возрастанию частот
  3. Пока в очереди больше одного узла:
    • Извлечь два узла с наименьшими частотами
    • Создать новый узел с частотой, равной сумме частот извлеченных узлов
    • Назначить извлеченные узлы левым (0) и правым (1) потомками нового узла
    • Поместить новый узел обратно в очередь
  4. Полученное дерево задает коды: путь от корня к символу дает кодовое слово

Пример из учебника: По каналу связи передаются сообщения, содержащие только семь букв: П, И, Ф, А, Г, О, Р. Для букв П, И, Ф используются такие кодовые слова: П – 1, И – 011, Ф – 001. Укажите кратчайшее кодовое слово для буквы А.

Решение:

  1. Строим двоичное дерево кодов
  2. Находим свободные ветки
  3. Выбираем кратчайшее кодовое слово

Ответ: 000

7 Задание №7: Кодирование информации

Основные формулы:

Тип информации Формула Обозначения
Графическая I = P · i P - количество пикселей, i - глубина цвета
Звуковая I = k · i · v · t k - каналы, i - глубина, v - частота, t - время
Видео I = P · i · v · t P - размер кадра, v - частота кадров
Передача I = V · t V - скорость, t - время

Единицы измерения:

  • 1 байт = 8 бит
  • 1 Кбайт = 1024 байт
  • 1 Мбайт = 1024 Кбайт
  • 1 Гбайт = 1024 Мбайт
  • 1 Кбит = 1000 бит
  • 1 Мбит = 1000 Кбит

Пример из учебника: Марафонцы решили нарисовать капибар. Сколько байт будет весить картина с капибарами размером 120 × 60 пикселей? В палитре используется 283 цвета.

Решение:

  1. Определяем глубину цвета: 2⁸ < 283 < 2⁹ ⇒ i = 9 бит
  2. Вычисляем количество пикселей: P = 120 × 60 = 7200
  3. Вычисляем объем: I = 7200 × 9 = 64800 бит = 8100 байт

Ответ: 8100

8 Задание №8: Комбинаторика

Основные формулы комбинаторики:

Тип Формула Описание
Размещения с повторениями A_n^k = n^k Каждый элемент может участвовать многократно
Размещения без повторений A_n^k = n!/(n-k)! Каждый элемент используется только один раз
Перестановки P_n = n! Все элементы различны, порядок важен
Сочетания C_n^k = n!/((n-k)!·k!) Все элементы различны, порядок не важен

Решение через Python:

from itertools import * # Размещения с повторениями for x in product('ABC', repeat=3): print(''.join(x)) # Перестановки for x in permutations('ABC'): print(''.join(x)) # Сочетания for x in combinations('ABCD', 2): print(''.join(x))

Пример из учебника: Алексей составляет пятизначные числа в восьмеричной системе счисления. Цифры могут встречаться в числе несколько раз или не встречаться совсем. Сколько различных чисел может составить Алексей?

Решение:

  1. На первое место можно поставить 7 цифр (кроме 0)
  2. На остальные 4 места - по 8 цифр
  3. Всего: 7 × 8 × 8 × 8 × 8 = 28672

Ответ: 28672

Программное решение на Python:

from string import punctuation f = open('text.txt') # Читаем текст из файла text = f.read() for symbol in punctuation: # Заменяем знаки препинания на пробелы text = text.replace(symbol, ' ') words = text.split() # Разделяем текст на слова count = 0 # Считаем вхождения слова "пример" for word in words: if word == 'пример': count += 1 print(count)

11 Задание №11: Кодирование информации

Основные понятия:

  • Бит - наименьшая единица информации (0 или 1)
  • Байт - 8 бит, кодирует один символ
  • Мощность алфавита - количество символов в алфавите
  • Информационный объем - количество бит для хранения информации

Формулы:

  • N = 2^i (мощность алфавита)
  • I = K * i (информационный объем текста)
  • K = (I + Iextra) * n (общий вес всех паролей)

Алгоритм решения:

  1. Определяем мощность алфавита N
  2. Находим количество бит на символ: i = ⌈log₂N⌉
  3. Находим количество бит на пароль: V = i * k
  4. Переводим в байты (округляем вверх)
  5. Учитываем дополнительную информацию
  6. Находим общий вес всех паролей

Пример из учебника: При регистрации в компьютерной системе каждому пользователю выдается пароль, состоящий из 11 символов. В качестве символов используют прописные и строчные буквы латинского алфавита (в нем 26 символов), а также десятичные цифры. В базе данных для хранения сведений о каждом пользователе отведено одинаковое и минимально возможное целое число байт. При этом используют посимвольное кодирование паролей, все символы кодируют одинаковым и минимально возможным количеством бит. Кроме собственно пароля, для каждого пользователя в системе хранятся дополнительные сведения, для чего выделено 13 байт на одного пользователя. В компьютерной системе выделен 1 Кбайт для хранения сведений о пользователях. О каком наибольшем количестве пользователей может быть сохранена информация в системе?

Решение:

  1. Мощность алфавита: N = 26 + 26 + 10 = 62
  2. Бит на символ: 2^5 ≤ 62 ≤ 2^6 ⇒ i = 6 бит
  3. Бит на пароль: 11 * 6 = 66 бит
  4. Байт на пароль: 66/8 = 9 байт (округляем вверх)
  5. Байт на пользователя: 9 + 13 = 22 байта
  6. Количество пользователей: 1024 / 22 = 46 (округляем вниз)

Ответ: 46

Законы алгебры логики:

  • Закон де Моргана: ¬(x ∨ y) = ¬x ∧ ¬y, ¬(x ∧ y) = ¬x ∨ ¬y
  • Дистрибутивный закон: x ∧ (y ∨ z) = (x ∧ y) ∨ (x ∧ z)

Пример из учебника: Обозначим через ДЕЛ(n, m) утверждение «натуральное число n делится без остатка на натуральное число m». Для какого наибольшего натурального числа A логическое выражение ¬ДЕЛ(x, A) → (ДЕЛ(x, 28) → ¬ДЕЛ(x, 49)) истинно при любом натуральном значении переменной x?

  1. Записываем выражение: ¬A → (28 → ¬49)
  2. Раскрываем импликацию: A ∨ (¬28 ∨ ¬49)
  3. Упрощаем: A ∨ ¬28 ∨ ¬49
  4. Выражение должно быть истинно при любом x
  5. Находим, когда ¬28 ∨ ¬49 = 0: 28 ∧ 49 = 1
  6. Наименьшее общее кратное 28 и 49: НОК(28, 49) = 196

Решение:

  1. Находим общую формулу: F(n) = 3^(n-1) * n!
  2. Вычисляем:

    • F(2023) = 3²⁰²² * 2023!

    • = 3² * 2023 * 1013 = 9 * 2023 * 1013

Программное решение на Python:

from functools import lru_cache @lru_cache(None) def f(a): if a >= 12: # Условие выигрыша return 0 moves = [f(a+2), f(a*2)] # Все возможные ходы win_moves = [i for i in moves if i <= 0] # Выигрышные ходы для текущего игрока if win_moves: return -max(win_moves) + 1 else: return -max(moves) # Поиск начальных значений для выигрыша Пети первым ходом for i in range(1, 12): if f(i) == 1: print(f"Петя выигрывает первым ходом при S = {i}") # Поиск начальных значений для выигрыша Вани первым ходом for i in range(1, 12): if f(i) == -1: print(f"Ваня выигрывает первым ходом при S = {i}")

Особенности разных задач:

  • Задача 19: Обычно требуется найти минимальное S для выигрыша Вани
  • Задача 20: Обычно требуется найти S для выигрыша Пети вторым ходом
  • Задача 21: Обычно требуется найти S, при котором Ваня выигрывает вторым ходом

Примеры ответов:

  • Задача 19: Минимальное S = 14
  • Задача 20: Минимальное S = 5
  • Задача 21: Максимальное S = 11

Решение задачи на две кучи:

from functools import lru_cache @lru_cache(None) def f(a, b): if a + b >= 20: # Условие выигрыша return 0 # Все возможные ходы для двух куч moves = [f(a+2, b), f(a+3, b), f(a, b+2), f(a, b+3)] win_moves = [i for i in moves if i <= 0] if win_moves: return -max(win_moves) + 1 else: return -max(moves) # Пример: первая куча = 13 камней, ищем S для второй кучи for s in range(1, 20): if f(13, s) == 1: print(f"Петя выигрывает первым ходом при S = {s}") break

Советы по решению:

  1. Всегда сначала решайте задачу "вручную" для понимания логики
  2. Используйте кэширование (@lru_cache) для ускорения программы
  3. Внимательно читайте условие - операции могут быть разными
  4. Для задач с неудачными ходами меняйте логику выбора хода
